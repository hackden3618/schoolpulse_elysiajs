# Common Module

This directory contains shared utilities, configurations, types, and middleware used across the entire backend application.

## Structure

*   `config/`: Application configuration (e.g., environment variables).
*   `errors/`: Custom error classes and error handling logic.
*   `middleware/`: Elysia middleware functions.
*   `types/`: Global TypeScript types and interfaces.
*   `utils/`: Helper functions (e.g., hashing, string manipulation).
