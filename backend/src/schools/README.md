# Schools Module

## Purpose
Manages school entities and their core details.

## Responsibilities
- Create and manage schools.
- Expose RESTful endpoints for school management.

## Dependencies
- `PrismaClient` (Database access)

## Public API
- `GET /schools`: List all schools.
- `POST /schools`: Create a new school.

## Future Work
- Add update and delete school endpoints.
- Integration with billing/subscriptions.
